## Installation

Install utilfunctions with npm

```bash
  npm install utilfunctions
```

## Usage

```js
import { isNumber, isString, isRegExp, isBoolean, isUndefined } from "utilfunctions";

  function test(number){
     return isNumber(number); 
  }

```
